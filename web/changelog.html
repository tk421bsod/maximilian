Changelog

This is a collection of release notes.

*A quick disclaimer: While I try to include every change in a release, these changelogs may omit a few small changes. Look through Maximilian's commit history to get even greater detail on changes.


v0.6.0

released on 20 April 2021

  • Lots of changes to the startup process
  • Bug fixes and performance improvements for the music extension
  • A complete rework of todo lists
  • discord.py 1.7.1
  • Improvements to selfhosting

v0.5.2

released on 18 March 2021

  • You can now specify a logging level through command line arguments
  • 'Added to queue' messages look better
  • Major restructuring of core code for easier maintenance
  • New command that removes items from the song queue
  • Total duration now shows hours

v0.5.0

released on 28 February 2021

  • New `images` extension, with a set of commands that apply effects to avatars
  • Better error handling for missing permissions
  • Bug fixes for the music extension
  • Performance improvements for the music extension, allowing for playback without gaps
  • Improvements to todo addition/deletion messages
  • New `rolemembers` command that shows people with the specified role
  • Added the ability to delete your entire todo list (`todo deleteall`)
  • New `source` command that shows source code for the specified command
  • New `stop` command that stops music

v0.4.0

released on 10 February 2021

  • Many new features for the music extension
  • A few minor internal improvements to custom commands

v0.3.0

released on 23 January 2021

  • New extension -- music!
  • Added support for playing music from Youtube
  • Songs are added to a queue if you try to play songs while one's already playing

v0.1.0

released on 21 November 2020

  • Misc. bug fixes
  • Added version to status
  • Made Maximilian public
  • New help command
  • Organized code into modules

pre-rewrite

before september 2020

No releases are documented before this point. Maximilian was first brought online on 7 September 2019 and was completely rewritten from 16 April 2020 to early September 2020.
