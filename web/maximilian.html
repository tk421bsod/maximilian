Maximilian

A Discord bot with some cool features that does some cool stuff. (currently under development)

Features

Customizable Prefix

You can customize Maximilian's prefix to prevent it from conflicting with other bots.

Reaction Roles

You can give users roles (certain sets of permissions) when they react to a message.

User Info

You can get information about a certain user, like their account age, permissions, roles, and status.
